# FeelSoGood — 임채훈 공식 웹사이트

> **www.feelsogood.shop**  
> AI 기술력과 군 장교 출신 리더십을 겸비한 경영 컨설턴트 임채훈의 공식 웹사이트

---

## 🎯 프로젝트 목표

www.sociotoons.com 의 정보 구조와 전환 중심 구성 방식을 참고하여,
임채훈 대표의 강의/컨설팅 역량을 소개하는 **전환(Conversion) 최적화 랜딩 페이지**를 구현합니다.

---

## ✅ 현재 구현된 기능

### 페이지 구성
| 섹션 | 내용 |
|------|------|
| Hero | 풀스크린 히어로, 통계 카운터 애니메이션 |
| About | 대표 소개 (경력·수상·저서) |
| Services | 6개 전문 서비스 카드 |
| 강의 클래스 | 6개 강의 과정 카드 (동적 렌더링) |
| 경력·자격 | 타임라인, 자격증, 군 경력 |
| 강의 이력 | 필터 탭 (전체/공공기관/대학/기업) + 동적 카드 |
| Quote | 임팩트 인용구 섹션 |
| Contact | 문의 폼 (DB 저장) |
| Footer | 링크, 사업자 정보 |

### 기술 기능
- ✅ 반응형 디자인 (모바일/태블릿/데스크탑)
- ✅ 스크롤 애니메이션 (Intersection Observer)
- ✅ 숫자 카운터 애니메이션
- ✅ 햄버거 모바일 메뉴
- ✅ 헤더 스크롤 변화 효과
- ✅ 강의이력 필터링 탭
- ✅ 문의 폼 (테이블 API 저장)
- ✅ 맨 위로 버튼

## 📁 파일 구조

```
index.html          # 메인 페이지
css/
  style.css         # 전체 스타일 (반응형 포함)
js/
  main.js           # 인터랙션, 데이터 렌더링
README.md
```

## 🌐 진입 URI

| 경로 | 설명 |
|------|------|
| `/` or `/index.html` | 메인 랜딩 페이지 |
| `/#about` | 대표 소개 섹션 |
| `/#services` | 서비스 섹션 |
| `/#lectures` | 강의 클래스 섹션 |
| `/#credentials` | 경력·자격 섹션 |
| `/#clients` | 강의 이력 섹션 |
| `/#contact` | 문의 섹션 |

## 🗄️ 데이터 모델

### `contact_inquiries` 테이블
| 필드 | 타입 | 설명 |
|------|------|------|
| id | text | 고유 ID (자동) |
| name | text | 문의자 성함 |
| org | text | 소속 기관/회사 |
| email | text | 이메일 |
| phone | text | 연락처 |
| type | text | 문의 유형 |
| message | rich_text | 문의 내용 |
| submitted_at | datetime | 제출 일시 |

## 🚀 추천 다음 개발 단계

1. **실제 사진 업로드** — 임채훈 대표 프로필 사진 추가
2. **기관 로고 추가** — 강의 이력 섹션에 실제 기관 로고 배치
3. **카카오채널 연동** — 문의하기 버튼 카카오 채널 링크 추가
4. **블로그/유튜브 연동** — SNS 링크 실제 URL 연결
5. **포트폴리오 페이지** — 강의자료 샘플, 성과물 별도 페이지
6. **수강후기 섹션** — 실제 수강생/기관 담당자 후기 추가
7. **SEO 최적화** — Open Graph, 구조화 데이터 추가
8. **Google Analytics** — 방문자 분석 코드 삽입
